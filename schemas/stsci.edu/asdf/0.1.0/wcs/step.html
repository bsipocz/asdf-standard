Type: object.

Describes a single step of a WCS transform pipeline.

Properties:

name

Type: string.

The name of this step on the WCS transform pipeline. This name may be used by a library to allow the user to say “transform from pixel space to the focal plane” without performing the entire WCS pipeline.

axes

Type: array of ( axis ).

A description of each of the input axes for this step.

Items:

Type: axis.

transform

Type: transform.

The transform from this step to the next one. The last step in a WCS should not have a transform, but exists only to describe the frames and units of the final output axes.
